## 📌 Sobre o projeto

Este projeto reúne as atividades desenvolvidas ao longo da disciplina de Desenvolvimento Web II.  
O objetivo é praticar conceitos fundamentais de desenvolvimento web, com foco em:

- PHP (estrutura, sintaxe e organização)
- Manipulação de formulários
- Conexão com banco de dados usando PDO
- Sistema de autenticação com sessões
- Operações CRUD (Create, Read, Update, Delete)
- Organização e refatoração de código
