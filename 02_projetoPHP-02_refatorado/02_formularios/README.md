# Aula 04 DWII — Formulários, GET, POST e Validação

Este diretório contém a evolução do portfólio pessoal, agora com foco em interatividade, processamento de dados e segurança utilizando PHP puro. 

O grande destaque desta etapa é a criação de um sistema de formulário de contato funcional, implementando boas práticas de back-end.


## ⚙️ Como executar

Abra o terminal na pasta raiz do seu projeto e inicie o servidor embutido do PHP:

cd ~/workspaces/2026-DWII
php -S localhost:8001
